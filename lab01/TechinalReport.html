  • About Password:

    I set a password by JavaScript to protect my web page. I set JavaScript at the front of the body tag, and I set up a loop within it. The code will get the password from the user through the pop-up dialog box of the website. Only when the password is correct, it will break away from the loop and execute the subsequent HTML code.

  • New Tag:

    I learned HTML by myself in the summer vacation, so I use my favorite combination of two tags: input and button here. The input tag allows users to enter the value they want to feedback, while the button tag can be used to submit data. At the same time, I use placeholder inside the input to further prompt the user what kind of data to enter.
